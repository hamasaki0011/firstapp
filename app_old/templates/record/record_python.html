(1) Python 3のセットアップ

UbuntuにはPython3がプレインストールされている。 搭載されているPython3のバージョンは、 APT(Advanced Packaging Tool)のaptコマンドで、 システムを更新およびアップグレードした後で確認する。

$sudo apt update

$sudo apt -y upgrade

ここで指定した"-y"フラグは、すべてのアイテムがインストール されることに同意することを示す。 しかし、Linuxのバージョンによってはシステムを更新・アップグレード中に 追加で同意を求められるかもしれない。 プロセス完了後、次のように入力してシステムにインストールされている Python 3のバージョンを確認する。 バージョン番号が端末ウィンドウに出力される。 表示されるバージョン番号は様々だが、2022年10月現在は次のようになる。

$python3 -V

Python 3.10.6

Pythonのソフトウェアパッケージを管理するには、開発プロジェクトで使用する プログラミングパッケージをインストールおよび管理するツールであるpipを インストールする必要がある。

$sudo apt install -y python3-pip

pipは、Pythonのパッケージ（「ライブラリ」や「モジュール」とも呼ぶ）を インストール、管理するためのパッケージ。 pipが依存関係を管理してくれるので、ユーザーはパッケージの依存関係を一切 気にする必要がない。 これにより、以下のように依存関係を気にすることなくパッケージを取り扱うことが できる。

$pip3 install package_name

ここで、package_nameとして、Web開発用のDjangoや科学計算用のNumPyなど、任意の Pythonパッケージまたはライブラリを参照することができる。 NumPyをインストールする場合は、コマンド pip3 install numpyを実行する。 また、pythonには堅牢なプログラミング環境セットアップのため、インストールした方が 良いパッケージと開発ツールがいくつかある。

$sudo apt install -y build-essential libssl-dev libffi-dev python3-dev

ここでインストールするパッケージは以下になる。

build-essentialパッケージ 基本的な開発ツール
libssl-devパッケージ OpenSSLのヘッダファイルとライブラリ Webシステム開発では、ユーザーPerlを使うが、 HTTP通信をサポートするためには、IO::Socket::SSLを インストールする必要がある。
libffi-devパッケージ FFI(Foreign function interface)の機能が提供される。 ある言語から異なる言語の関数を呼び出すための仕組み。 ex)pythonからc/c++など
