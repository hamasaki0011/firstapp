Python3のインストール

UbuntuにはPython3がプレインストールされている。
搭載されているPython3のバージョンは、APT(Advanced Packaging Tool)のaptコマンドで、 システムの更新とアップグレードを行った後で確認することができる。 

    以下にその例を示す。

    $sudo apt update
    $sudo apt -y upgrade

    ここで指定した"-y"フラグは、すべてのアイテムがインストールされることに同意することを示す。
    ただし、Linuxのバージョンによってはシステムを更新とアップグレード中に追加で同意を求められることもある。
    プロセス完了後、次のように入力してシステムにインストールされているPython3のバージョンを確認する。
    バージョン番号が端末ウィンドウに出力される。表示されるバージョン番号は様々だが、2022年10月現在は次のようになる。

    $python3 -V
    Python 3.10.6

    Pythonのソフトウェアパッケージを管理するには、開発プロジェクトで使用するプログラミングパッケージを
    インストールおよび管理するツールであるpipをインストールする必要がある。

    $sudo apt install -y python3-pip

    pipは、Pythonのパッケージ（「ライブラリ」や「モジュール」とも呼ぶ）をインストール、管理するためのパッケージ。
    pipが依存関係を管理してくれるので、ユーザーはパッケージの依存関係を一切気にする必要がない。
    これにより、以下のように依存関係を気にすることなくパッケージを取り扱うことができる。

    $pip3 install package_name

    ここで、package_nameとして、Web開発用のDjangoや科学計算用のNumPyなど、任意のPythonパッケージまたは
    ライブラリを参照することができる。
    NumPyをインストールする場合は、コマンド pip3 install numpyを実行する。
    また、pythonには堅牢なプログラミング環境セットアップのため、インストールした方が
    良いパッケージと開発ツールがいくつかある。

    $sudo apt install -y build-essential libssl-dev libffi-dev python3-dev

インストールするパッケージをまとめると以下にようになる。

表 Python3関連ライブラリー
パッケージ名 機能概要
build-essential 基本的な開発ツール
libssl-dev OpenSSLのヘッダファイルとライブラリ。
Webシステム開発では、HTTP通信をサポートするためには、IO::Socket::SSLをインストールする必要がある。
libffi-dev FFI(Foreign function interface)の機能が提供される。 ある言語から異なる言語の関数を呼び出すための仕組み。
ex)pythonからc/c++など
